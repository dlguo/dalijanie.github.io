我们的故事

大力 Dali

大力出生在安徽省马鞍山市。其实大力原来是他的小名，但是他嫌原来的名字太难写了，就求爸爸妈妈换了现在的这个大力。2002年夏天在外婆家，他看了世界杯上小罗那个离谱的任意球后成为了一位追风⚽️少年。他小时候的梦想就是成为职业球员，在世界杯决赛伤停补时被替换上场打进绝杀进球带领🇨🇳队夺冠。后来他一不小心成了网瘾少年，两本订阅全年的杂志就是《电脑爱好者》和《中国国家地理》。⚽️和💻一直影响着他让他成为了现在这个大力。

菁爽 Janie

Indiana的烈日寒风

大力和菁爽2015年夏天入学了Purdue，大力读的是Industrial Engineering，菁爽在Mathematics。他们刚来学校的时候都会去教会蹭吃蹭喝结识朋友。那天朋友攒了一个局，大力刚刚踢完球换了双鞋澡都没洗就去了，当天晚上大力就开始发动攻势（<你还玩篮球啊！好厉害 <哈哈 本科的时候打着玩），整整三个半月大力借抽象代数教材，计划Northern Michigan赏枫🍁road trip，学游泳🏊‍♀️，剃头👩‍🦲，租pickup🛻去机场接机，经历了九九八十一难才在那年的12月28号正式在一起了。

短短的PhD和长长的Covid

PhD的日子又辛苦又快乐，大力和菁爽6年里跑了好多好多地方（比如Alaska🌌，Wisconsin🛥️，Barcelona🇪🇸，Colorado🏂，New England🍁，Oregon🏔️，Grand Canyon🌄等等等等），认识了好多好多朋友。好在大力和菁爽的advisor都超棒的，他们都很顺利地毕业🎓了！在最倒霉的Covid期间他们又运气很好的一起来到了Seattle工作。来到Seattle之后，他们又去了很多很多地方被Covid和工签先后困住😭没法远行。不过他们认识了更多更多的好朋友，他们也渐渐的在Seattle安定了下来。

求婚

2022年冬天大力见证了他从13岁看到30岁的梅西带领🇦🇷拿到了世界杯冠军。与此同时，他也在Hawaii偷偷计划着求婚。他借了朋友的信用卡，用小号手机号和小号Email把场地、材料都预备整齐，在12月28号也就是七年Anniversary的时候在Kirkland的Marina Park叫了朋友们摆好了场地打算惊喜求婚。那天大力先找朋友约着吃饭，中午的时候谎称oncall去办公室加班没法参加晚餐。然后再由朋友谎称散步，带到湖边的求婚场地（谁知道西雅图的冬天不下雨我的运气是多么的好😭）。

感谢好朋友们从下午开始就一直在帮大力吹气球布置场地😭😭😭一切准备就绪，大力接着扯谎说加班好累直接回家了，当时菁爽路过看到求婚还想去凑热闹，直到大力走到眼前了才看到求婚对象竟是她自己😂大力当时自信满满觉得自己提前背好的词绝对不可能忘记🙅‍♂️结果当众打脸，菁爽一个爆哭大力不仅忘词忘的一干二净，连捧花💐都忘记交给菁爽。整个流程磕磕巴巴也算是没忘了带戒指💍，结果当然是一个大大的
